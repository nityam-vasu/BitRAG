About BitRAG
BitRAG - Chat with your Documents

A RAG (Retrieval-Augmented Generation) system for chatting with PDF documents using LLMs.

Features:
  • PDF document indexing and retrieval
  • RAG-powered question answering
  • Multiple LLM model support via Ollama
  • Session-based document management
Keyboard Shortcuts:
  • Enter - Send message
  • Ctrl+L - Clear chat
